# Користувацькі мітки

* Автори: Kefas Lungu

Примітка: для цього додатка потрібна NVDA 2025.1 або новіша версія.
## Огляд

Користувацькі мітки — це додаток NVDA, який дозволяє додавати власні мітки до елементів керування без міток та редагувати наявні. Це особливо корисно для програм із кнопками чи елементами керування, які не мають міток або NVDA не може їх правильно визначити.

## Особливості

* Призначайте користувацькі мітки елементам керування без міток
* Редагуйте наявні користувацькі мітки
* Видаляйте користувацькі мітки, коли вони більше не потрібні
* Керуйте всіма мітками через панель налаштувань
* Мітки зберігаються для кожної програми окремо для кращої організації, а також для експорту та імпорту з метою подальшого спільного використання

## Підтримувані типи керування

Можна позначити наступні типи керування:

* Кнопки
* Кнопки меню
* Перемикачі
* Прапорці
* Радіокнопки
* Комбіновані списки
* Повзунки
* Вкладки
* Елементи меню

## Жести

* NVDA+Control+L: встановити або редагувати користувацьку мітку для поточного елемента керування
* NVDA+Control+Delete: видалити користувацьку мітку з поточного елемента керування
* NVDA+Control+J: перевірити, чи має поточний елемент керування користувацьку мітку
* NVDA+Control+; відкрити налаштування користувацьких міток

## Використання

### Встановлення користувацької мітки

1. Сфокусуйтесь на елементі керування, якому потрібно надати мітку
2. Натисніть NVDA+Control+L
3. З’явиться діалог з інформацією про елемент керування
4. Введіть бажану мітку в текстове поле
5. Натисніть OK, щоб зберегти мітку

### Редагування наявної мітки

1. Сфокусуйтесь на елементі керування, який має користувацьку мітку
2. Натисніть NVDA+Control+L
3. Змініть мітку в текстовому полі
4. Натисніть OK, щоб зберегти зміни

### Видалення мітки

Видалити мітку можна двома способами:

1. Сфокусуйтеся на елементі керування та натисніть NVDA+Control+Delete
2. Або відкрийте діалог мітки (NVDA+Control+L) і натисніть кнопку Видалити

### Керування всіма мітками

1. Натисніть NVDA+Control+; щоб відкрити панель налаштувань користувацьких міток
2. Перегляньте мітки, упорядковані за програмою
3. За потреби використовуйте кнопки «Редагувати», «Видалити», «Видалити програму» або «Видалити все»

## Панель налаштувань

Доступ до панелі налаштувань користувацьких міток можна отримати за допомогою:

* Комбінації клавіш NVDA+Control+;
* Меню NVDA > Параметри > Налаштування > Користувацькі мітки

На панелі відображаються всі користувацькі мітки, упорядковані у вигляді дерева за програмами. Ви можете:

* **Редагувати**: змінити вибрану мітку
* **Видалити**: видалити вибрану мітку
* **Видалити програму**: видалити всі мітки для вибраної програми
* **Видалити все**: видалити всі користувацькі мітки

## Зберігання

Мітки зберігаються у файлах JSON у каталозі конфігурації NVDA в папці `customLabels`. Кожна програма має власний файл JSON, що спрощує створення резервних копій або обмін мітками для конкретних програм.
